❤️ Heart Disease Prediction (Machine Learning + Streamlit)

https://heartdiseaseprediction-rq34.streamlit.app

A simple end-to-end machine learning project that predicts the likelihood of heart disease using patient health data.
This project includes:
	•	Data preprocessing
	•	Model training (Logistic Regression, Random Forest, KNN, SVM)
	•	Saving the best model
	•	A Streamlit web app for interactive prediction
	•	Deployment on Streamlit Cloud
